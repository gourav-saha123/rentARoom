# Geo Location Based Rental Platform (MERN)
A location based house rental platform where property owners can register and list homes, and
tenants can discover nearby properties using realtime geospatial search.

## Tech Stack
1. MongoDB — Database with GeoJSON + 2dsphere indexing
2. Express.js — Backend API
3. React — Frontend UI
4. Node.js — Server runtime

## Features
1. Owners can register and list rental properties
2. Tenants can search for nearby homes based on current location
3. Distance based filtering using MongoDB geospatial queries
4. Results sorted by proximity

## Best Frontend Options for Geolocation
1. Google Maps JavaScript API — Most complete, reliable, production grade
2. Mapbox GL JS — Highly customizable and visually modern
3. Browser Geolocation API — Get user’s current coordinates
